# 项目名称

## 项目概述

这是一个完整的全栈项目，包含后端、前端和移动端三个部分，实现了资产管理、采购管理、文件管理和AI分析等功能。项目采用现代化的技术栈，提供了完整的企业级应用解决方案。

## 技术栈

### 后端 (moon-backend)
- **语言**: Go
- **框架**: Ent ORM, GraphQL (gqlgen)
- **数据库**: PostgreSQL
- **认证**: JWT
- **存储**: S3兼容存储
- **其他**: 分层架构, 领域驱动设计

### 前端 (moon-frontend)
- **框架**: Angular 21
- **UI库**: NG-ZORRO
- **状态管理**: Apollo Client
- **API**: GraphQL
- **构建工具**: Angular CLI

### 移动端 (moon-mobile)
- **框架**: Ionic v8 + Angular
- **UI**: Ionic原生组件
- **构建工具**: Ionic CLI
- **部署**: Capacitor

## 核心功能

### 1. 资产管理
- **资产分类管理**: 创建、编辑、删除资产分类
- **资产类型管理**: 管理不同类型的资产
- **资产管理**: 资产的全生命周期管理，包括创建、编辑、删除、查询
- **资产状态跟踪**: 实时跟踪资产状态和位置

### 2. 采购管理
- **采购计划管理**: 制定和管理采购计划
- **采购计划类型管理**: 管理不同类型的采购计划
- **采购验收管理**: 管理采购验收流程
- **采购专家管理**: 管理采购专家信息
- **采购风险分析**: 分析采购风险
- **采购实施管理**: 管理采购实施过程
- **采购评审管理**: 管理采购评审流程

### 3. 文件管理
- **文件上传下载**: 支持文件的上传和下载
- **文件管理**: 文件的创建、编辑、删除、查询
- **文件存储**: 使用S3兼容存储进行文件存储
- **预签名URL**: 生成预签名URL用于安全的文件操作

### 4. AI分析功能
- **LLM模型管理**: 管理LLM模型配置
- **采购分析**: 使用AI进行采购分析
- **令牌使用管理**: 跟踪LLM令牌使用情况
- **风险评估**: 使用AI进行风险评估

### 5. 系统管理
- **用户管理**: 用户的创建、编辑、删除、查询
- **角色管理**: 管理用户角色
- **权限管理**: 管理系统权限
- **组织管理**: 管理组织信息

### 6. 移动端功能
- **采购管理**: 移动端采购记录管理
- **验收管理**: 移动端验收记录管理
- **扫码功能**: 支持扫码录入和查询
- **状态更新**: 实时更新记录状态
- **离线操作**: 支持离线操作和数据同步

## 系统架构

### 后端架构
- **分层架构**: 表示层、应用层、领域层、基础设施层
- **领域驱动设计**: 基于领域模型的设计
- **GraphQL API**: 提供灵活的API接口
- **S3集成**: 集成S3兼容存储

### 前端架构
- **组件化设计**: 基于Angular组件的模块化设计
- **服务层**: 封装API调用和业务逻辑
- **GraphQL客户端**: 使用Apollo Client管理数据
- **响应式设计**: 适配不同屏幕尺寸

### 移动端架构
- **Ionic框架**: 基于Ionic的移动应用框架
- **Angular集成**: 使用Angular进行业务逻辑开发
- **Capacitor**: 提供原生功能访问
- **移动端优化**: 针对移动设备的性能优化

## 部署说明

### 后端部署
1. 配置环境变量
2. 构建Go应用
3. 部署到服务器
4. 配置数据库连接
5. 启动服务

### 前端部署
1. 构建前端应用: `ng build --prod`
2. 部署到静态文件服务器
3. 配置API端点

### 移动端部署
1. 构建应用: `ionic build`
2. 生成平台代码: `ionic capacitor add ios/android`
3. 构建原生应用: `ionic capacitor build ios/android`
4. 部署到应用商店
